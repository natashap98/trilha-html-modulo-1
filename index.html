Estrutura Básica do HTML

(Voltar)

Uma página HTML típica começa com:

<!DOCTYPE html>
<html lang="pt-BR">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Página</title>
  </head>
  <body>
    <!-- Conteúdo visível -->
  </body>
</html>

<!DOCTYPE html>: informa ao navegador que o documento está em HTML5. Sem isso, pode haver comportamentos inesperados.
<html>: é o elemento raiz, que pode incluir o atributo lang para definir o idioma.
<head>: traz metadados como título, estilos e scripts.
<body>: contém todo o conteúdo visível.

Tags e Elementos

(Voltar)

Uma tag é um nome entre os símbolos < >, como <p>, <img>, <li> e outros.
Um elemento inclui a tag de abertura, o conteúdo e a tag de fechamento, por exemplo:
<p>Texto do parágrafo</p>
Há também tags de encerramento automático, ou seja, não precisam de fechamento, como <img> ou <br>.

Atributos das Tags

(Voltar)

Os atributos fornecem informações extras aos elementos, no formato chave="valor", e são colocados na tag de abertura.
Exemplos:

<a href="https://exemplo.com">Link</a>
<img src="foto.jpg" alt="Descrição da imagem">
<p id="paragrafo1" class="meu-estilo">Texto...</p>

href, src, id, class são atributos comuns.

Textos

(Voltar)

Tags de texto incluem:

<h1> a <h6> para títulos com diferentes níveis de importância;

<p> para parágrafos;

Listas: <ul> (usada quando a ordem dos itens não importa (ex: lista de compras, tópicos));
<ol> (usada quando a ordem dos itens importa (ex: passo a passo, rankings, instruções)).

Elementos Semânticos: main, header, footer

(Voltar)

header: representa a introdução de uma página ou seção. Pode conter títulos, logotipos, menus etc;
main: contém o conteúdo principal da página. Deve haver apenas um único <main> por documento;
footer: representa o rodapé de uma página ou seção, contendo informações complementares como autor, direitos autorais, links de navegação secundária etc.

Essas tags são importantes para dar significado semântico à página, ajudando navegadores, mecanismos de busca e tecnologias assistivas a entenderem melhor o conteúdo.
